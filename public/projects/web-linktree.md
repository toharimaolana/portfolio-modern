# Project: Linktree Clone (React)
Oleh: **Mohaamad Tohari Maolana**

---

## Tentang Proyek Ini

Proyek ini adalah *clone* dari layanan populer "Linktree", yang dibuat sebagai latihan untuk memperdalam keahlian **React**. Ini adalah *Single Page Application* (SPA) yang ringan dan cepat, berfungsi sebagai halaman pusat untuk semua tautan penting (media sosial, portofolio, dll) dalam satu URL.

## Fitur Utama

* **Daftar Tautan Dinamis:** Tautan (link) dimuat dari data (misal: file JSON) sehingga mudah di-update.
* **Desain Responsif:** Dibuat dengan pendekatan *mobile-first*, memastikan tampilan sempurna di ponsel.
* **Animasi Halus:** Menggunakan Framer Motion (atau CSS murni) untuk animasi *hover* pada tombol.
* **Komponen Modular:** Dibangun dengan komponen React yang dapat digunakan kembali (`<ButtonLink />`, `<ProfileHeader />`).
